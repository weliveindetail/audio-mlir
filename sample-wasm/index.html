audio-mlir samples in WebAssembly

Two DSP-MLIR kernels compiled to WASM (relocatable object → wasm-ld), driven from JS over the same memref ABI as the native sample-macOS.cpp host: a MemRefDescriptor1D in linear memory plus shared parameter globals.

A 440 Hz sawtooth fed through a 101-tap windowed-sinc low-pass FIR. The @cutoff global sweeps the corner frequency.
